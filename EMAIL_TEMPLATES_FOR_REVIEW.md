- **NO TABLES:** Tables are not allowed in the gallery space. Small bins or boxes on the floor are permitted for prints.

- **AGE POLICY:** The venue enforces a strict [ageRestriction] age policy.

**Attention Live Painters:** We love featuring live body painting and canvas work. If you'd like to paint live, let us know and we will get you details.

Thanks,

## PLACEHOLDER FIELDS REFERENCE

These fields will be automatically filled in when emails are sent:

- **[firstName]** - Applicant's first name (e.g., "Pricklyash")
- **[organizationName]** - Organization name (e.g., "Pancakes & Booze")
- **[eventLocation]** - Event city/location (e.g., "Denver")
- **[eventName]** - Full event name (e.g., "TEST - Summer Market 2026")
- **[eventVenue]** - Venue name (e.g., "Red Rocks Park and Amphitheatre")
- **[eventDate]** - Event date (e.g., "Wednesday, April 1, 2026")
- **[paymentDueDate]** - Early payment deadline (e.g., "Thursday, March 19, 2026")
- **[installDate]** - Installation date (e.g., "Tuesday, February 17, 2026")
- **[installStartTime]** - Installation start time (e.g., "8:00 AM")
- **[installEndTime]** - Installation end time (e.g., "10:00 AM")
- **[ageRestriction]** - Venue age policy (e.g., "All Ages" or "21+")

## NOTES FOR REVIEW

1. **Styling:** All emails use simple text formatting with gray divider lines - no fancy backgrounds or colors
2. **Unsubscribe link:** Automatically added at bottom of every email
3. **Generic fallback:** We kept the existing generic template for any other vendor categories
4. **Three email types:**
   - **Invitation Email** - Sent to vendors you want to invite to apply for your event
   - **Artist Application Received** - Auto-sent when an artist submits their application
   - **Vendor Table Application Received** - Auto-sent when a vendor submits their table application
